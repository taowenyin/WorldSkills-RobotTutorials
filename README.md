# WorldSkills-RobotTutorials
世界技能大赛案例

WIFI:172.16.0.1

USB:172.22.11.2

### 1、右轮
>* 右轮方向IO：A/DIO5、A/DIO6
>* 右轮PWM：A/PWM0
>* 右轮编码器：C/ENC1

### 2、左轮
>* 左轮方向IO：A/DIO7 A/DIO13
>* 左轮PWM：A/PWM1
>* 左轮编码器：C/ENC0

### 3、线性运动PID参数

8.000 0.001 0.000

### 4、旋转运动PID参数

0.120 0.000 0.001

### 5、上下舵机
>* 左轮PWM：A/PWM2
>* PWM：0.051-0.073
>* 频率：50Hz

### 6、小车头旋转
>* 头旋转方向IO：B/DIO5 B/DIO6
>* 头旋转PWM：B/PWM0
>* 头旋转编码器：A/ENC
